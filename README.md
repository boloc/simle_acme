# simple_acme
Acme.sh 域名证书一键申请脚本

此脚本可以帮助你快速使用acme.sh脚本申请域名的ssl证书，并且可以保存到你想要的位置

# 使用方法
```sh
wget -N https://cdn.jsdelivr.net/gh/boloc/simple_acme/README.md/simple_acme.sh && chmod +x simple_acme.sh && bash simple_acme.sh
```
